1.目次サンプル

今回作る目次はこちら。

目次

  2. ???????????????????????????
  3. ???????????????????????????
  4. ???????????????????????????

HTMLはこちら。

<div id="toc-container">
    <p class="toc-title">目次</p>
    <ol class="toc-list">
        <li><a href="#1">???????????????????????????</a></li>
        <li><a href="#2">???????????????????????????</a></li>
        <li><a href="#3">???????????????????????????</a></li>
        <li><a href="#4">???????????????????????????</a></li>
        <li><a href="#5">???????????????????????????</a></li>
        <li><a href="#6">???????????????????????????</a></li>
        <li><a href="#7">まとめ</a></li>
    </ol>
</div>

CSSはこちら。

#toc-container {
    background: white, none repeat scroll 0 0;
    border: 1px solid skyblue;
    display: table;
    font-size: 95%;
    margin: 0 auto 0 auto;
    padding: 10px;
    width: auto;
}

.toc-title {
    font-weight: 700;
    text-align: center;
}

#toc-container li,
#toc-container ol,
#toc-container ol li {
    list-style-position: outside;
    margin-left: 1em;
    margin-right: 1em;
    padding-left: 0;
}

2.HTMLの解説

まず、HTMLのコードの解説から。

<div>で全体を覆います。idを目次全体という意味の『toc-container』にします。

次に<p>で目次タイトルを書きます。idを目次のタイトルという意味の『toc-title』にします。

次に<ol>と<li>で番号付けリストを書きます。目次の中身ですね。それぞれでリンクするようにします。もし複数ページのサイトを作っていて，例えば2ページ目に#3のリンク先がある場合，

<a href=”aiueo-2.html#3”>

といったようにすれば繋がります。

リンク先の方では単一ページの場合と変わらずに

<h2 id="3">3.???????????????????????????</h2>

とすればいいです。

3.CSSの解説

次に，CSSの解説です。目次全体の『#toc-container』について。

『border』は枠線を1pxで1本線(solid)でスカイブルー色(skyblue)で表示します。

『margin: 0 auto 0 auto;』で目次を中央に表示します。上下の空白を0，左右をオートにしています。

『padding』で10pxの空白を枠線内側に表示します。

『width』は幅をオート（auto）表示します。

『.toc-title』は『font-weight: 700;』で目次タイトル名太字にし，『text-align: center;』で中央に来るようにしています。

『#toc-container li,#toc-container ol,#toc-container ol li』では『list-style-position: outside;』で1や2などのリストの番号が左端に固定されるようにしています。『inside』だとリストの頭につき位置がバラバラになります。『margin-left: 1em;』や『margin-right: 1em;』でリストの左右の位置を調整します。『padding-left: 0;』で間隔の調整をします。
